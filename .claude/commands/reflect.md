# /reflect - Fehler-Reflexion

Analysiere vergangene Fehler und leite Erkenntnisse ab.

## Anweisungen

Führe den `reflect` Skill aus:

1. **Memory lesen:**
   - `memory/ERRORS.md` - Dokumentierte Fehler
   - `memory/PATTERNS.md` - Erkannte Muster
   - `memory/LEARNINGS.md` - Bisherige Erkenntnisse

2. **Analysieren:**
   - Welche Fehler gab es zuletzt?
   - Gibt es neue Muster?
   - Welche Learnings sind noch offen?

3. **Trends erkennen:**
   - Häufige Fehler identifizieren
   - Verbesserungen/Verschlechterungen
   - Offene Präventionsmaßnahmen

4. **Skills ableiten:**
   - Welche Learnings können automatisiert werden?
   - Skill-Vorschläge machen
   - Bei guter Idee: `skill-from-learning` nutzen

5. **Dokumentieren:**
   - Reflexion in LEARNINGS.md festhalten
   - Neue Erkenntnisse notieren

Output: Reflexions-Report mit Erkenntnissen und Empfehlungen.
